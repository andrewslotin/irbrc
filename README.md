# .irbrc

## Installation
```git clone git://github.com/andrewslotin/irbrc.git
gem install wirble pretty_print hirb
ln -s irbrc/.irbrc ~/.irbrc```

## Includes:

* [Wirble](http://pablotron.org/software/wirble/)
* [Hirb](http://tagaholic.me/hirb/)

## Helper methods

* ```clear``` to cleanup screen
